# 📂 File Handling

A simple and interactive **File Handling Application** built using **Python** and **Streamlit**. This project allows users to create, read, update, delete, and manage text files through a clean web interface.

---

## 🚀 Features

* 📝 Create new text files
* 📖 Read existing files
* ✏️ Update file content
* 🗑️ Delete files
* 📋 View all available files
* 🎨 Clean and responsive user interface
* ⚡ Built with Streamlit for fast interaction

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **HTML/CSS**
* **File Handling (CRUD Operations)**

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/amber-arya/File-Handling.git
```

Go to the project folder:

```bash
cd File-Handling
```

Install dependencies:

```bash
pip install streamlit
```

Run the application:

```bash
streamlit run app.py
```
